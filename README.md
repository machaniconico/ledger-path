# Ledger Path

日商簿記3級の仕訳を、毎日少しずつ練習する日本語Webアプリです。添付仕様の段階リリース方針に沿い、最初の版は「今日の学習 → 仕訳入力 → 決定論的採点 → 復習」の縦切りに絞っています。

> 日本商工会議所の公式サービスではありません。問題・解説・採点は独自の学習用で、合格を保証するものではありません。

## 現在の機能

- 3問のオリジナル仕訳演習
- 全角数字、桁区切り、円記号を正規化するBigIntベースの採点
- 借貸不一致、科目、金額、借貸逆転を区別したフィードバック
- 誤答の復習キューと進捗表示
- 初回の学習時間設定と、この端末への進捗保存
- ヒントと即時解説を使わないミニ模試モード
- キーボード操作、明確なフォーカス、モバイル表示、reduced motion対応

進捗はブラウザの `localStorage` にだけ保存されます。認証、サーバー同期、公式試験相当の模試、2級・1級カリキュラムは後続フェーズです。

## 開発

Node.js `>=22.13.0` が必要です。

```bash
npm install
npm run dev
npm test
npm run lint
```

`npm test` はデプロイ用ビルド、採点ゴールデンテスト、サーバーレンダリング検証を実行します。
