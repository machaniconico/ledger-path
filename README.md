# Ledger Path

日商簿記3級の仕訳を、毎日少しずつ練習する日本語Webアプリです。「今日の学習 → 仕訳入力 → 決定論的採点 → 復習」を軸に、カテゴリ別の演習・SRS復習・本番形式の模試までを1画面で回せます。

> 日本商工会議所の公式サービスではありません。問題・解説・採点は独自の学習用で、合格を保証するものではありません。

## 現在の機能

- **55問・11カテゴリの問題バンク** — 現金・預金／商品売買（三分法）／債権・債務／手形・電子記録／貸付・借入／固定資産／給与・税金／資本・剰余金／経過勘定／訂正仕訳／その他・貸倒れ。すべて借方合計＝貸方合計で内部整合し、独立レビューで会計的正確性を検証済み。
- **カテゴリ別ブラウズ／フィルタ** — カテゴリごとの進捗を見ながら、強化したい論点を選んで練習。
- **答えを漏らさない仕訳入力** — 各行の借方／貸方は学習者が選択（正解の行構成を提示しない）。行は2〜8行で増減でき、勘定科目は資産／負債／純資産／収益／費用でグループ化（`optgroup`）した69科目から選ぶ。
- **BigIntベースの決定論的採点** — 全角数字・桁区切り・円記号を正規化。借貸不一致／科目／金額／借貸逆転を区別したフィードバックと即時解説。
- **軽量スペースドリピティション復習** — 回答履歴から Leitner 風の優先度で復習キューを並べ替え。
- **連続学習日数（ストリーク）とデイリーゴール** — トップに当日の進捗を表示。
- **カテゴリ別成績ダッシュボード** — カテゴリごとの正答率と弱点をアクセシブルに可視化。
- **本番形式ミニ模試** — カテゴリ配分で最大15問を抽選、経過時間を計測、ヒント・即時解説なしで一括採点し、カテゴリ別内訳と誤答レビュー導線を表示。
- **この端末への進捗保存** — 認証・サーバー同期なしで `localStorage` にのみ保存（v1→v2 は非破壊マイグレーション）。
- **アクセシビリティ** — キーボード操作、明確なフォーカス、モバイル表示、`prefers-reduced-motion` 対応。

進捗はブラウザの `localStorage` にだけ保存されます。認証、サーバー同期、2級・1級カリキュラムは後続フェーズです。

## 開発

Node.js `>=22.13.0` が必要です。

```bash
npm install
npm run dev
npm test
npm run lint
```

`npm test` はデプロイ用ビルドに加え、採点ゴールデンテスト、問題バンクの整合性検証、学習進捗・模試ロジックの単体テスト、サーバーレンダリング検証を実行します。

## 構成

- `app/` — Next.js（vinext）アプリ本体（`LedgerPathApp.tsx` が学習UI）。
- `lib/questions.ts` — 型付きの問題バンク（`CATEGORIES` / `ACCOUNT_GROUPS` / `ALL_ACCOUNTS`）。
- `lib/scoring.ts` — 仕訳の正規化・決定論的採点。
- `lib/progress.ts` — SRS復習順・連続日数・デイリーゴールの純関数（`now` 注入で決定論的）。
- `lib/mock.ts` — 模試の出題選定（seed注入）と採点集計の純関数。
- `tests/` — `node --test` によるロジック／SSR検証。
- Cloudflare Workers 上で動作（`worker/index.ts`）。
